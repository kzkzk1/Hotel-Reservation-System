# Hotel-Reservation-System
ソフトウェア工学 ホテル予約システムをUMLでオブジェクト指向分析・設計し，Javaで部分実装する。

## 進め方
- Ex1　ドメイン分析：問題領域　→　クラス図 + オブジェクト図
- Ex2　要求分析：　機能3つ以上について，　ユースケース図 + ユースケース記述 + アクティビティ図
- Ex3　システム分析：　コラボレーション図  + クラス図

## 注意点
.astaファイルはバイナリ形式で，2人が同時に触れないらしいです。
